# Challenge Amigo Secreto ONE

Este proyecto es una implementación del challenge "Amigo Secreto" de ONE (Oracle Next Education G8) utilizando JavaScript, HTML y CSS.

Puedes utilizar esta app [haciendo click aqui]()

La prorgamación JavaScript y parte del css y html fueron hechos por Emmanuelito18.

## Descripción

El challenge "Amigo Secreto" es una actividad en la que el usuario ingresa una lista de nombres de sus amigos, y la aplicación elige de entre todos los amigos inrgesados uno al azar y lo muestra en pantalla.

## Características

- Asignación aleatoria de amigos secretos.
- Interfaz de usuario simple y fácil de usar.
- Validación para asegurar que nadie deje la lista de amigos vacía

## Tecnologías utilizadas

- HTML
- JavaScript
- CSS
